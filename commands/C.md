# C Commands

## cat

Displays file contents.

Example

```bash
cat file.txt
```

---

## cd

Change directory.

Example

```bash
cd Downloads
```

---

## chmod

Modify file permissions.

Example

```bash
chmod +x script.sh
```

---

## chown

Change file ownership.

Example

```bash
chown user:user file.txt
```

---

## clear

Clear terminal screen.

Example

```bash
clear
```

---

## cp

Copy files.

Example

```bash
cp file.txt backup.txt
```

---

## curl

Transfer data from servers.

Example

```bash
curl https://example.com
```
