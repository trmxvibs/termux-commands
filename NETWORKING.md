# Networking Commands

## ping

Check connectivity.

Example

```bash
ping google.com
```

---

## nmap

Network scanner.

Example

```bash
nmap 192.168.1.1
```

---

## netstat

Display network connections.

Example

```bash
netstat -tulpn
```

---

## ssh

Remote login.

Example

```bash
ssh user@192.168.1.10
```
