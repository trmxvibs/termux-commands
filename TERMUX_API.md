# Termux API Commands

Install API:

```bash
pkg install termux-api
```

---

## Battery Status

```bash
termux-battery-status
```

---

## Camera

```bash
termux-camera-photo photo.jpg
```

---

## Vibrate

```bash
termux-vibrate -d 1000
```

---

## Notification

```bash
termux-notification -t "Hello" -c "Script Finished"
```
