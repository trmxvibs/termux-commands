# Termux Quick Cheat Sheet

A quick reference guide for the most commonly used Termux and Linux commands, sorted from A to Z.

| Command | Description |
| :--- | :--- |
| **alias** | create custom shortcuts |
| **apt** | package manager (Debian base) |
| **awk** | text and data processing |
| **bash** | execute shell scripts |
| **bc** | command-line calculator |
| **bg** | resume paused job in background |
| **cat** | view file contents |
| **cd** | change directory |
| **chmod** | change file permissions |
| **clear** | clear the terminal screen |
| **cp** | copy files or directories |
| **curl** | fetch or download web data |
| **date** | show current date and time |
| **df** | show free disk space (`-h` for readable) |
| **dpkg** | install local `.deb` packages |
| **du** | show directory space usage |
| **echo** | print text to terminal |
| **env** | show environment variables |
| **exit** | close current terminal session |
| **find** | search for files and folders |
| **free** | show RAM/memory usage |
| **git** | version control system |
| **grep** | search text for specific patterns |
| **head** | output the first 10 lines of a file |
| **history**| show previously typed commands |
| **htop** | interactive process monitor |
| **id** | show user and group IDs |
| **ifconfig**| show network IP information |
| **jobs** | list active background tasks |
| **kill** | stop a running process by ID |
| **less** | view file contents page by page |
| **ls** | list files and directories (`-la` for all) |
| **mkdir** | create a new folder/directory |
| **mv** | move or rename files |
| **nano** | simple text editor |
| **netstat**| display network connections |
| **nmap** | scan networks and open ports |
| **ping** | test network connectivity |
| **pkg** | Termux primary package manager |
| **ps** | view currently running processes |
| **pwd** | print working (current) directory |
| **rm** | delete files (`-rf` for folders) |
| **scp** | securely copy files over SSH |
| **sed** | find and replace text in files |
| **ssh** | secure remote login |
| **tail** | output the last 10 lines of a file |
| **tar** | create or extract archive files |
| **top** | view system resource usage |
| **touch** | create a new empty file |
| **uname** | print system information (`-a` for all) |
| **unzip** | extract `.zip` files |
| **uptime** | show how long the system has been running |
| **vim** | advanced text editor |
| **wc** | count lines, words, and characters |
| **wget** | download files directly from the web |
| **whoami** | show current username |
| **xargs** | build and execute command lines |
| **yes** | output a string repeatedly |
| **zip** | compress files into an archive |

---
**Termux Special Commands**

| Command | Description |
| :--- | :--- |
| **termux-setup-storage** | grant storage permission to Termux |
| **termux-wake-lock** | prevent device from sleeping |
| **termux-wake-unlock** | allow device to sleep normally |
| **termux-reload-settings**| apply changes made to config files |
