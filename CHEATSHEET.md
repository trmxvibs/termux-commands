# Termux Cheat Sheet

| Command | Description |
|------|------|
| ls | list files |
| cd | change directory |
| pwd | show location |
| rm | delete files |
| mkdir | create folder |
| cp | copy files |
| mv | move files |
| chmod | change permissions |
| wget | download files |
| curl | fetch web data |
| ping | test connectivity |
| ssh | remote login |
| tar | archive files |
